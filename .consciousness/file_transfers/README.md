## How to Use

### Sending a File:
1. Copy file to this directory
2. Create a `.meta` file with description:
   ```
   filename.txt.meta:
   FROM: Computer 1
   TO: Computer 2
   PURPOSE: Stripe API key for revenue system
   CREATED: 2025-11-03T08:30:00Z
   ```
3. Commit and push
4. Add note in target computer's inbox

### Receiving a File:
1. Pull latest changes
2. Check this directory for new files
3. Read associated `.meta` file for context
4. Use file as needed
5. Delete file and push (to keep directory clean)
